# bench/data

Synthetic prompt corpus, key fixtures, and tool-call corpus used by the
benchmark scenarios. Generated by deterministic scripts (added in P01) so
anyone can reproduce the same workload.

Files arriving in P01:

- `prompts.jsonl` ~10,000 synthetic prompts across micro, small, medium,
  large, xlarge buckets.
- `keys.json` 1,000 Marg API keys across 100 teams across 10 routes,
  power-law distribution.
- `tool_calls.jsonl` ~500 synthetic tool calls, used by Kavach gate
  evaluations in P08.

P00 ships this directory empty so the scaffold is complete.
